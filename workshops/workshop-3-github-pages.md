# Workshop 3: Publish Your Site Online

## Put Your Website Live on GitHub Pages

**Duration:** 1.5 - 2 hours  
**Difficulty:** Beginner-Intermediate  
**What you'll learn:** GitHub basics, version control, publishing websites online

---

## What We're Building

Your website will be live on the internet at a URL like: `yourusername.github.io/my-website`

---

## Step 1: Create a GitHub Account

1. Go to [github.com](https://github.com)
2. Click "Sign up"
3. Choose a username (you can use your name or something creative)
4. Enter your email and create a password
5. Verify your email address

**Tip:** Your username will be part of your website URL, so choose wisely!

---

## Step 2: Install GitHub Desktop (Easier Way)

1. Go to [desktop.github.com](https://desktop.github.com)
2. Download GitHub Desktop for your computer
3. Install it and sign in with your GitHub account

**Alternative:** If you prefer, you can use the website, but GitHub Desktop is easier for beginners.

---

## Step 3: Create a New Repository

1. Open GitHub Desktop
2. Click "File" → "New Repository"
3. Name it: `my-first-website` (or any name you want)
4. Choose where to save it on your computer
5. Add a description: "My first website for Island Code for Locals"
6. Click "Create Repository"

---

## Step 4: Add Your Files

1. In GitHub Desktop, click "Show in Finder" (Mac) or "Show in Explorer" (Windows)
2. Copy your HTML and CSS files into this folder
3. Go back to GitHub Desktop
4. You should see your files listed under "Changes"
5. Write a message at the bottom: "First website upload"
6. Click "Commit to main"

---

## Step 5: Push to GitHub

1. Click "Publish repository" (or "Push origin" if you already published)
2. This uploads your files to GitHub
3. Wait for it to finish

---

## Step 6: Enable GitHub Pages

1. Go to [github.com](https://github.com) in your web browser
2. Find your repository (click on your username, then the repository name)
3. Click on "Settings" tab (at the top)
4. Scroll down to "Pages" in the left sidebar
5. Under "Source", select "Deploy from a branch"
6. Choose "main" branch and "/ (root)" folder
7. Click "Save"

---

## Step 7: Your Website is Live!

1. Wait 1-2 minutes for GitHub to build your site
2. Go back to the Settings → Pages section
3. You'll see a link like: `https://yourusername.github.io/my-first-website`
4. Click the link - your website is online!

**Share this link with friends and family!**

---

## Step 8: Update Your Website

When you want to make changes:

1. Edit your HTML or CSS files on your computer
2. Open GitHub Desktop
3. You'll see the changes listed
4. Write a message about what you changed
5. Click "Commit to main"
6. Click "Push origin"
7. Wait 1-2 minutes - your website updates automatically!

---

## Common Issues and Solutions

**Problem:** "Repository not found"
- **Solution:** Make sure you're signed in to GitHub Desktop

**Problem:** "Can't find Pages settings"
- **Solution:** Make sure you're looking at YOUR repository, not someone else's

**Problem:** Website shows "404 Not Found"
- **Solution:** Wait a few minutes, or make sure your `index.html` file is in the main folder

**Problem:** Changes don't appear
- **Solution:** Make sure you clicked "Push origin" and wait 1-2 minutes

---

## Tips

- **Your website URL:** `https://yourusername.github.io/repository-name`
- **Free forever:** GitHub Pages is free for personal websites
- **Update anytime:** Just edit files and push changes
- **Multiple websites:** You can create many repositories for different websites
