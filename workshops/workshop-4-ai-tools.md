# Workshop 4: Intro to AI Tools

## Write Posts, Translate, and Design Using ChatGPT

**Duration:** 1.5 - 2 hours  
**Difficulty:** Beginner  
**What you'll learn:** Using AI to help create content, translate text, and generate ideas

---

## What We're Learning

How to use ChatGPT and other AI tools to:
- Write website content
- Translate between Thai, Burmese, and English
- Generate ideas for your website
- Get help with coding questions

---

## Step 1: Get Started with ChatGPT

1. Go to [chat.openai.com](https://chat.openai.com)
2. Sign up for a free account (or use Google/Apple to sign in)
3. You'll see a chat interface

**Free vs Paid:**
- Free version: Good for learning and basic tasks
- Paid version: Faster and more features (optional)

---

## Step 2: Write Website Content

**Example 1: Write an About Page**

Type this in ChatGPT:
```
Write a short "About Us" paragraph for a café in Koh Phangan, Thailand. 
Make it friendly and welcoming. Keep it under 100 words.
```

ChatGPT will write content for you!

**Example 2: Write Menu Descriptions**

Type:
```
Write descriptions for a café menu:
- Coffee
- Fresh mango smoothie
- Pad Thai

Make each description 1-2 sentences, appealing and appetizing.
```

---

## Step 3: Translate Content

**Translate to Thai:**

Type:
```
Translate this to Thai: 
"Welcome to Sunset Café. We serve fresh coffee and amazing food."
```

**Translate to Burmese:**

Type:
```
Translate this to Burmese:
"We are open daily from 8am to 8pm. Visit us near Haad Rin Beach."
```

**Translate to English:**

Type:
```
Translate this Thai text to English:
[Paste your Thai text here]
```

---

## Step 4: Generate Ideas for Your Website

**Ask for ideas:**

Type:
```
Give me 5 ideas for sections on a yoga studio website in Koh Phangan.
```

**Ask for color suggestions:**

Type:
```
Suggest a color palette for a beach café website. 
Give me 3-4 colors that work well together.
```

**Ask for content ideas:**

Type:
```
Give me 10 ideas for blog posts about living in Koh Phangan.
```

---

## Step 5: Get Help with Code

**Ask coding questions:**

Type:
```
How do I change the background color of my website header to blue?
```

**Debug errors:**

Type:
```
My CSS isn't working. Here's my code:
[Paste your code]

What's wrong?
```

**Learn new concepts:**

Type:
```
Explain what CSS "flexbox" is in simple terms with examples.
```

## Best Practices

**Good prompts:**
- Be specific about what you want
- Mention your audience (e.g., "for locals in Koh Phangan")
- Ask for tone (e.g., "friendly", "professional", "casual")
- Specify length (e.g., "50 words", "2 paragraphs")

**Example of a good prompt:**
```
Write a welcoming homepage paragraph for a Thai restaurant website. 
Target audience: tourists visiting Koh Phangan. 
Tone: warm and inviting. 
Length: 3-4 sentences.
```

## Important Reminders

**Always check AI responses:**
- AI can make mistakes
- Always review and edit what AI generates
- Make sure translations are correct (ask a native speaker if possible)
- AI is a helper, not a replacement for your own creativity

**Be respectful:**
- Don't use AI to write fake reviews
- Don't copy content from other websites
- Use AI to help you create YOUR original content

## Advanced Tips

**Better prompts:**
- "Act as a web designer and suggest..."
- "In the style of..."
- "For beginners, explain..."
- "Give me 3 options for..."

**Combine AI with your work:**
- Use AI for ideas, then write your own version
- Use AI to translate, then edit for local context
- Use AI to learn, then practice yourself
