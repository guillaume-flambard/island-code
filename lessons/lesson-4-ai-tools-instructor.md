# Instructor Lesson Plan: Lesson 4
## Intro to AI Tools - ChatGPT and Beyond

---

## Overview & Learning Objectives

### Workshop Overview
Students will learn to use AI tools (primarily ChatGPT) to help create website content, translate text, generate ideas, and assist with coding. This lesson introduces AI as a helpful assistant tool, not a replacement for learning.

### Learning Objectives
By the end of this workshop, students will be able to:
- Create a ChatGPT account
- Write effective prompts for AI
- Use AI to generate website content
- Translate content between Thai, Burmese, and English
- Get coding help from AI
- Generate ideas for websites
- Understand AI limitations and best practices

### Prerequisites
- Completed Lessons 1-3 (HTML, CSS, GitHub Pages)
- Basic understanding of websites
- Email address for ChatGPT account
- Access to internet

### Duration
**Total Time:** 90-120 minutes

- "Easy to sign up"
- "Used by millions"

1. Go to chat.openai.com

### Part 3: Introduction to Prompts (15 minutes)

**What is a Prompt?**
- "Prompt = question or instruction for AI"
- "Like telling assistant what to do"
- "Better prompt = better answer"

**Talking Points:**
- "AI reads your prompt"
- "Gives answer based on prompt"
- "Clear prompt = clear answer"

**Good vs Bad Prompts:**

**Bad Prompt:**
- "Write something"
- "Make it good"
- "Help me"

**Why Bad:**
- Too vague
- No details
- AI doesn't know what you want

**Good Prompt:**
- "Write a short 'About Us' paragraph for a caf√© in Koh Phangan. Make it friendly and welcoming. Keep it under 100 words."

**Why Good:**
- Specific topic
- Clear audience
- Specific length
- Tone specified

**Prompt Formula:**
1. **What** - What do you want?
2. **Who** - For whom?
3. **How** - What style/tone?
4. **How much** - Length/details?

**Practice Together:**
- Write prompt for menu item
- Write prompt for email
- Write prompt for social media post

**Check Understanding:**
- "What makes a good prompt?"
- "Try writing one yourself"

### Part 8: Best Practices & Limitations (10 minutes)

**Review AI Responses (5 min)**

**Talking Points:**
- "Always check AI's work"
- "AI can make mistakes"
- "Edit and improve"

**What to Check:**
- Does it make sense?
- Is it accurate?
- Does it match your style?
- Is it appropriate?

**Understand Limitations (3 min)**

**AI Limitations:**
- Can make mistakes
- Doesn't know everything
- May give outdated info
- Can't replace human judgment

**When to Use:**
- First drafts
- Ideas generation
- Translations (with checking)
- Learning help

**When NOT to Use:**
- Final content without review
- Critical information
- Personal decisions
- Legal/medical advice

**Ethical Use (2 min)**

**Do:**
- Use AI to help you create
- Edit and improve AI content
- Add your own voice
- Check accuracy

**Don't:**
- Copy AI content without editing
- Use for fake reviews
- Plagiarize from others
- Rely completely on AI

**Talking Points:**
- "AI helps you create YOUR content"
- "You're still the creator"
- "Add your personal touch"

## Common Issues & Solutions

### Issue: Can't create ChatGPT account
**Symptoms:** Signup fails, verification issues
**Solutions:**
- Try different email
- Use Google/Apple sign-in
- Check internet connection
- Wait and retry
- Use phone number instead

### Issue: AI gives wrong answer
**Symptoms:** Incorrect information, wrong translation
**Solutions:**
- Rephrase prompt more clearly
- Ask follow-up question
- Check with other sources
- Edit AI response
- Understand AI can make mistakes

### Issue: AI response too long/short
**Symptoms:** Not right length
**Solutions:**
- Specify length in prompt ("50 words", "2 paragraphs")
- Ask to shorten/lengthen
- Edit response yourself

### Issue: Translation not accurate
**Symptoms:** Translation sounds wrong
**Solutions:**
- Check with native speaker
- Try different wording
- Check context
- Ask AI to rephrase
- Edit manually

### Issue: AI doesn't understand prompt
**Symptoms:** Wrong or confusing response
**Solutions:**
- Make prompt more specific
- Break into smaller parts
- Provide more context
- Try different wording
- Give examples

### Issue: Too dependent on AI
**Symptoms:** Student only using AI, not learning
**Solutions:**
- Emphasize AI as helper
- Encourage editing
- Ask them to explain concepts
- Require personal additions
- Review together

## Differentiation Strategies

### For Beginners
- **Template prompts:** Give ready-made prompts
- **Simple tasks:** Translate one sentence
- **Guided practice:** Step-by-step with help
- **Focus on basics:** Content generation only

**Modifications:**
- Start with simple prompts
- Use AI for inspiration only
- Focus on one task at a time
- Provide prompt templates

### For Intermediate Students
- **Complex prompts:** Multi-step instructions
- **Independent use:** Less guidance
- **Help others:** Become prompt helpers
- **Advanced features:** Use AI for coding, design

**Extensions:**
- Chain prompts (AI output ‚Üí new prompt)
- Use AI for planning entire website
- Generate multiple options
- Learn prompt engineering

### For Advanced Students
- **Teaching:** Help explain to others
- **Advanced AI:** Use different AI tools
- **Automation:** Use AI for repetitive tasks
- **Integration:** Use AI in workflow

**Advanced Topics:**
- Prompt engineering techniques
- Using multiple AI tools
- AI-assisted development
- Custom AI solutions

### For Language Barriers
- **Visual prompts:** Show examples
- **Translation first:** Use AI to translate instructions
- **Simple language:** Use basic prompts
- **Native language:** Allow prompts in their language
