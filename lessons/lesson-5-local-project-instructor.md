# Instructor Lesson Plan: Lesson 5
## Local Project - Complete Website for Local Business

---

## Overview & Learning Objectives

### Workshop Overview
This is the capstone project where students combine all skills learned (HTML, CSS, GitHub Pages, AI tools) to create a complete website for a real local business in Koh Phangan. This workshop emphasizes project planning, execution, and presentation.

### Learning Objectives
By the end of this workshop, students will be able to:
- Plan a complete website project
- Gather information from a business
- Create a multi-section website
- Apply all learned skills together
- Test and improve their website
- Publish a complete project online
- Present their work to others

### Prerequisites
- Completed Lessons 1-4 (HTML, CSS, GitHub Pages, AI Tools)
- Basic understanding of all concepts
- Completed website files from previous lessons

### Duration
**Total Time:** 120 minutes (can be split into 2 sessions)

### Part 3: Build Phase (40 minutes)

**Create HTML Structure (15 min)**

**Talking Points:**
- "Start with structure"
- "Like Lesson 1 - HTML first"
- "Don't worry about styling yet"

**Action:**
- Students create HTML file
- Add all sections
- Use semantic HTML: `<header>`, `<main>`, `<section>`, `<footer>`

**Template to Start:**
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>[Business Name] - Koh Phangan</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>[Business Name]</h1>
        <nav>
            <a href="#home">Home</a>
            <a href="#about">About</a>
            <a href="#menu">Menu</a>
            <a href="#contact">Contact</a>
        </nav>
    </header>
    
    <main>
        <section id="home">...</section>
        <section id="about">...</section>
        <section id="menu">...</section>
        <section id="contact">...</section>
    </main>
    
    <footer>...</footer>
</body>
</html>
```

**Walk Around:**
- Help with structure
- Answer questions
- Check progress

**Style with CSS (15 min)**

**Talking Points:**
- "Now make it beautiful!"
- "Like Lesson 2 - add colors, fonts"
- "Match your business theme"

**Action:**
- Students create CSS file
- Link to HTML
- Add colors, fonts, spacing
- Style each section

**Key Styles:**
- Header styling
- Section spacing
- Menu list styling
- Footer styling
- Responsive design

**Walk Around:**
- Help with CSS
- Suggest improvements
- Troubleshoot issues

**Add Content (10 min)**

**Talking Points:**
- "Fill in real information"
- "Use AI-generated content"
- "Make it personal"

**Action:**
- Add gathered information
- Use AI to help write
- Translate if needed
- Add personal touches

**Walk Around:**

## Common Issues & Solutions

### Issue: Can't choose business
**Symptoms:** Student unsure what to build
**Solutions:**
- Suggest local business they know
- Use template business idea
- Partner with classmate
- Build personal portfolio site

### Issue: Too much to do
**Symptoms:** Overwhelmed by project scope
**Solutions:**
- Simplify to 3 sections only
- Use templates as starting point
- Focus on completing, not perfecting
- Break into smaller tasks

### Issue: Information gathering difficult
**Symptoms:** Can't get business info
**Solutions:**
- Use AI to generate realistic info
- Look at similar businesses online
- Use template information
- Create imaginary but realistic business

### Issue: Website not coming together
**Symptoms:** Sections don't work together
**Solutions:**
- Start with template
- Focus on one section at a time
- Use consistent styling
- Ask for help

### Issue: Not enough time
**Symptoms:** Can't finish in time
**Solutions:**
- Focus on core sections
- Use templates
- Simplify design
- Can continue after workshop

### Issue: Presentation anxiety
**Symptoms:** Scared to present
**Solutions:**
- Optional to present
- Can share URL only
- Small groups
- Supportive environment

## Differentiation Strategies

### For Beginners
- **Use template:** Start with business-page-template
- **Simplified scope:** 3 sections only
- **Focus on completion:** Not perfection
- **Extra help:** One-on-one support
- **Pair programming:** Work with helper

**Modifications:**
- Pre-written HTML structure
- CSS template to customize
- Simplified requirements
- Focus on publishing successfully

### For Intermediate Students
- **Full project:** All sections
- **Customization:** Original design
- **Advanced features:** Add contact form
- **Help others:** Become mentor

**Extensions:**
- Multiple pages
- Advanced CSS (animations, transitions)
- JavaScript basics
- Custom domain

### For Advanced Students
- **Complex features:** Contact forms, maps
- **Multiple pages:** Full website
- **Advanced CSS:** Grid, animations
- **Teaching:** Help others

**Advanced Topics:**
- JavaScript interactivity
- Form handling
- SEO basics
- Performance optimization

### For Language Barriers
- **Visual planning:** Use diagrams
- **Templates:** Ready-to-use content
- **AI translation:** Translate content
- **Support:** Extra help available
